PUYOPUYO CDDA from YSSCSICD.SYS Patch

Instruction:
(1) Make a Towns OS V2.1 L20 boot floppy disk with YSSCSICD.SYS installed, and copy (overwrite) the following files to the boot floppy disk.
  AUTOEXEC.BAT
  CONFIG.SYS
  PATCH.EXP
(2)Delete HCOPY, SYSINIT, and T_TOOL directories of the boot floppy disk.
(3)Do NOT write protect the floppy disk.  Insert PuyoPuyo CD-ROM in the SCSI CD-ROM.  Then boot from the floppy disk created in step (1).
(4)Wait until the program starts. (It automatically creates a patched executable and run.)
(5)Write protect the floppy disk.





PUYOPUYO YSSCSICD.SYSでBGMを再生するパッチ

使い方:
(1) YSSCSICD.SYSをインストールした、Towns OS V2.1 L20起動フロッピーディスクを作って、次のファイルをコピー(上書きコピー)する。
  AUTOEXEC.BAT
  CONFIG.SYS
  PATCH.EXP
(2)起動フロッピーディスクのHCOPY, SYSINIT, T_TOOLディレクトリを削除。
(3)フロッピーディスクライトプロテクトを解除した状態で、ぷよぷよ for FM TOWNSのCD-ROMをSCSI CD-ROMドライブに入れて、フロッピーディスクから起動。
(4)ぷよぷよが起動するまで待つ。
(5)起動フロッピーディスクのライトプロテクトをかける。
