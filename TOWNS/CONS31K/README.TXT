CONS31K.COM
by CaptainYS http://www.ysflight.com

Run this command to set CRTC in 31KHz mode in the Command Mode, ORICON, or F-BASIC Console on FM TOWNS.

Also if the .EXP program crashed and the screen mode is messed up, this command will reset CRTC.



FM TOWNSのコマンドモード、ORICON、F-BASICコンソールなどで実行することで、CRTCをコマンドモードだけど31KHzの状態に設定することができます。24KHzにシンクしないモニタでもこのコマンドで表示できるようになります。

また、.EXPプログラムのクラッシュなどの影響でスクリーンモードが変になったときもこのコマンドで復活させることができます。
